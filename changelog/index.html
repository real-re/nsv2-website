        标签说明

        beta：代表该版本为测试版本（不稳定）。

        移动端：指安卓和iOS平台。

        PC端：指Windows、Linux和macOS平台。

        重要说明

        本游戏只在Re的B站 _____Re、 官网及火影战记官方QQ群内发布更新。

        如果你是从其他渠道下载本游戏，均为盗版，

        请确保不要被他人 欺骗 或其他方式损失你的钱财，

        如果你的设备出现任何异常将与本团队无关。

        v2.1.4 [beta] 2022-8-？¶

        预告

        • 🔥新特性

          • 支持iOS（越狱），具体查看平台说明
          • UI：移动端的技能按键布局适配宽屏设备
          • UI：支持设置攻击按键大小

        • 🚀变更

          • 平衡忍具【查克拉手里剑】 增加3%查克拉获取加成至18%
          • 平衡忍具【忍具陷阱】 随等级成长增加伤害，500 * 等级
          • 平衡忍具【忍术大全】 普通技能的冷却时间若大于10则减少5秒，反之减少4秒
          • 平衡忍具【特质饭团】 回复血量降低1500至4500
          • 平衡忍具【锁链忍铠】 附加血量上限从6000降至4000
          • 平衡贡献忍者鸣人-九尾查克拉模式技能1 CD从10增加至15秒
          • 平衡贡献忍者阿斯玛技能1 CD从10增加至15秒
          • 平衡贡献忍者鬼鲛技能1 CD从10增加至15秒
          • 平衡贡献忍者千代婆婆技能1 CD从10增加至15秒
          • 平衡井野奥义2 控制的角色死后复活时间从3增加至5秒
          • 平衡猿飞日斩奥义2 初始复活时间从3增加至5秒，且从4级起逐级递增2秒
          • 平衡迪达拉奥义2 初始复活时间从3增加至5秒，且从4级起逐级递增1秒
          • 平衡长门技能1 每复活一名角色，将消耗30%最大生命值，若血量低于15%则直接阵亡
          • 平衡长门技能1 角色使用后导致阵亡时，将视为正常阵亡（取消快速复活）
          • 平衡角都技能3 减伤效果从25%降至20%
          • 平衡千代婆婆技能1 移除霸体效果
          • 平衡千代婆婆技能1 减伤效果从25%降至10%，具体查看角色说明
          • 平衡千代婆婆奥义1 移除霸体效果
          • 平衡1v1模式，角色复活时间从10增加至12秒
          • 角色到达4级时，复活时间从11增加至12秒
          • 角色到达5级时，复活时间从15增加至17秒
          • 角色到达6级时，复活时间从20增加至23秒

        • ⚡️优化

          • 💥安卓端的更新提示将不再采取强制更新策略
          • 低性能设备上游戏加载完成后的卡顿现象
          • 鬼鲛 AI在技能3的状态下残血撤退时，将优先放空此技能以恢复移动速度
          • 千代婆婆和勘九郎的AI优先使用技能1释放傀儡
          • 触发彩蛋后的对局数据将纳入统计范畴
          • UI：完善4v4结算界面显示
          • UI：支持选择查看已经购买的忍具

        • 🐞修复

          • 💥修复安卓端在非中文环境下无法进行游戏
          • 💥修复鹿丸技能3 影子手触发的崩溃
          • 💥修复纲手技能3 蛞蝓触发的崩溃
          • 💥修复牙、小李、勘九郎和千代婆婆在解锁技能时，可能导致技能遮罩被移除
          • 修复克隆模式下鹿丸技能3 可能导致其他鹿丸提前结束技能3的影子束缚状态
          • 修复4v4结算界面角色顺序错误
          • 修复鸣人觉醒没有限制 #85
          • 修复千代婆婆在傀儡阵亡后仍会受到减伤效果
          • 修复鬼鲛 AI处于技能3期间会使用其他普通技能
          • 修复AI购买【特质饭团】时会导致玩家的拉面冷却时间减少
          • 修复AI购买【特质饭团】后不会减少拉面的冷却时间
          • 修复玩家使用井野释放奥义2控制其他角色后会自动购买忍具
          • 修复可觉醒的角色在游戏结束后，结算击杀数会遗漏觉醒前的数据
          • 修复玩家经验条到99%时会消失
          • 修复玩家经验条在某些情况会显示为100%

        v2.1.3 [beta] 2022-7-11¶

        • 🚀变更

          • 安卓：最低支持版本从4.2.2提高至4.4
          • UI：制作团队界面增加了返回按钮（安卓端某些情况下使用返回手势会退到桌面）

        • ⚡️优化

          • 训练模式默认可自由选择角色
          • 移动速度提升了14%
          • PC端：默认居中显示游戏窗口
          • 牙 AI默认使用忍具陷阱取缔速走忍鞋
          • 小李和重吾的AI在闲置状态时优先使用强化类技能

        • 🐞修复

          • 💥修复一处导致闪退的根源
          • 修复游戏结算时极低概率出现绿色长条贴图（由上原因导致）
          • 修复自定义选择角色的第三个Ready文本位置错误
          • 修复几处技能和忍具的描述文本错误
          • 修复结算界面评分计算错误
          • 修复角都 AI达到2级时无法使用奥义1
          • 修复水门放置的苦无陷阱死亡后不会消失
          • 修复长门 AI在某些情况可重复使用奥义2
          • 修复角色被井野控制前，处于普通攻击状态时则无法被井野控制
          • 修复玩家被井野控制结束后，可能处于移动/攻击状态
          • 修复玩家被井野控制死后的视角问题
          • 修复小李、阿飞和鸣人-四尾模式移动停顿
          • 修复角色处于固定的移动状态后，使用速走忍鞋不会立即变更移动速度
          • 修复角色处于固定的移动状态时，速走忍鞋状态结束后不会立即变更移动速度

          • 修复角都达到2级且心脏为零时没有启用奥义1
          • 修复角都技能3 buff结束后仍存在反伤效果
          • 修复角都 AI无法通过技能1获得金币

          • 修复PC端在角色复活后极低概率不处于闲置状态

        v2.1.0 [beta] 2022-4-1¶

        • 🔥新特性

          • 支持 Windows 64位
          • PC端：支持F11键切换窗口大小
          • 安卓和Windows可以通过游戏内的News按钮访问游戏更新日志
          • UI：增加模式选择界面
          • 加入1v1、4v4和克隆模式
          • 加入旧版角色佩恩、大蛇丸和君麻吕（感谢萌姐（hausumasuta）提供AI）

        • 🚀变更

          • UI：部分界面PC端增加了返回按钮
          • PC端：移除箭头键控制方向

        • ⚡️优化

          • 优化角色和音频的预加载
          • 所有音频完全受静音选项控制
          • UI：降低部分界面动画的时长，减少等待时间

        • 🐞修复

          • 💥彻底终结安卓端音频问题
          • 修复Logo界面音频播放问题
          • PC端：修复一些键盘操作的问题
          • 修复小李、角都和干柿鬼鲛的技能问题

        • 🎉彩蛋

          • 多次点击Logo，进入克隆模式将会是4v4，部分模式可以自选角色
