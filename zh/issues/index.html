        问题汇总

        此栏目统计目前所有已经确认的问题，同时将罗列个版本反馈的问题及进度。

        1. 如有新问题，请在反馈社区中反馈。

        2. 如果你对哪条问题有疑义或补充，请反馈社区内相关条目下留言。

        3. 如发现哪条问题被移除时，则说明该问题重复或被合并到其他问题中，请注意翻阅查看。

        4. 如果已解决的问题在下一版本中复现，则需要重新提交反馈，并附上详细描述、原问题的序号（可选）。

        反馈的具体格式请参考下面的反馈问题格式。

        此栏目将不定时更新，建议早上阅读，因为通常会在晚上更新。

        最后更新: 2022年8月1日 20:00:36

        反馈问题格式

        反馈问题，如涉及到战斗，应涵盖以下部分：

        1. 游戏模式

        2. 对局角色：主要为你操作的角色（若可能受其他角色影响，应记录相应角色名）

        涉及到技能，请按照普攻（普通攻击）、技能1-3、奥义1-2格式进行填写（阿拉伯数字或中文数字皆可）。

        涉及到游戏内文字、技能描述等错误或不准确，请在反馈社区中增加新条目。

        若UI、贴图或角色产生偏移、错位等问题，最好提供错误时的截图以及简要描述。

        如需引用已存在的问题，请使用 # + 问题编号 格式（例如#15）。

        如问题难以描述，最好在反馈社区中贴上截图，或发送一段视频给bug反馈相关人员，并附上详细描述。

        做好反馈描述，不仅有利于开发者进行定位修复，也利于其他遇到相似问题的玩家参考和进行补充。

        目前以下bug影响较大，望周知

        截至目前，仍存在极个别特殊情况会导致游戏闪退，主要为引擎缺陷，等待后续逐步修复。

        如果你无法进行游戏或没有较好的体验，可以在火影战记的官方QQ群里找到其他官方成员的版本。

        已知限制¶

        PC端玩家请确保游戏你的游戏路径不包含中文

        否则可能会导致游戏闪退，此问题等待后续修复。

        PC端切换声音输出设备后没有声音

        此为引擎bug，目前上游仍未修复此问题，后续将尝试修复此问题。
